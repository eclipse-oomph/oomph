1  Select a Transport Type

Currently supported transport types are:

  • JVM to connect to an Workspace in the same Java Virtual Machine.
  • TCP to connect to an Installation by means of a SocketChannel.
  • SSL an extension to the TCP transport that adds TLS/SSL security.
  • HTTP to connect to an SetupTask that is made available by a servlet.
Continue with Add Configuration Parameters.
