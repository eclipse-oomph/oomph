The version of the Darkest Dark Theme feature installed in your IDE is not compatible with Eclipse 2025-06. If you update the IDE without also updating this feature, the IDE will fail to start and you will not be able to revert the update in the IDE.
We have worked on a solution to prevent such an update while version 2024.4.0.202412101111 is installed.
There now is a newer version 2025.1.0.202505261652 available that is compatible with Eclipse 2025-06 (as well as compatible with older versons) so you will be able to update both at once.
Use Help → Install New Software, enter the URL https://devstyle.codetogether.io/, ensure that Hide items that are already installed is unchecked, and choose DevStyle (includes Darkest Dark Theme) to update to the new version immediately.
